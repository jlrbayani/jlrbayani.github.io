Tetris

Tetris remade natively in Java. This was a project that I had complete control over programming, art, and music. I learned a lot about the game programming patterns, the game loop, and multithreading different pipelines for physics, rendering, and sound. This application also allows for you to save your high scores.

2D Platformer - Bloop

This is a 2D platformer made in javascript and HTML. It has simple physics and uses the base javascript rendering system. I also created a custom level editor for it that re-used game objects which made adding levels a lot easier.

Database Management

This was a project made predominantly in Typescript. I worked on the main implemention for handling a tree like structure for the search functionality of the application while my partner worked on the framework for the underlying database matrix to handle the information we were working with. The experience was a great way of learning how to balance a project with someone else. This also gave me a lot of exposure to Typescript, its promises functionality, and HTTP requests.

Polynomial Calculator

This application was written in C++ and was made to help with long Polynomial addition and multiplication. Using the command line and character ruleset given in the code, you can calculate almost any polynomial sum or product. This was implemented using a Linked List to handle longer polynomials.
